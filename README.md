# 🏠 HouseHunt – MERN Stack House Rental Application

## 📌 Project Overview

HouseHunt is a full-stack web application designed to simplify the process of finding and renting houses.
The platform allows users to register, log in securely, browse available rental properties, and add new properties for rent.

The project is built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)** and demonstrates full-stack development including authentication, database integration, API development, and responsive user interfaces.

---

## 🚀 Features

### User Features

* User Registration
* Secure Login using JWT Authentication
* Browse Available Properties
* Add New Rental Properties
* View Property Details

### System Features

* RESTful API using Express.js
* MongoDB Database Integration
* JWT Authentication & Protected Routes
* Full-stack MERN architecture
* Responsive and interactive UI

## ⚙️ How to Run the Project

### 1️⃣ Clone Repository

```
git clone https://github.com/your-username/HouseHunt-MERN.git
```

### 2️⃣ Install Backend Dependencies

```
cd backend
npm install
npm run dev
```

### 3️⃣ Install Frontend Dependencies

```
cd frontend
npm install
npm start
```
